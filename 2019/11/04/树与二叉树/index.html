树与二叉树

发表于2019-11-04|更新于2019-12-29
|阅读量:

树的练习和总结

二叉树的性质

1.在二叉树的第i层上至多有2^(i-1)个结点
2.深度为k的二叉树至多有2^k-1个结点

满二叉树和完全二叉树

3.对于任何一颗二叉树T,tade 终端结点树为n0，度为2的结点为n2则 n0 = n2 + 1
4.满二叉树深度为k且含有2^k-1个结点的二叉树，完全二叉树结点<满二叉树
5.满二叉树每一层上的结点树都是最大结点数，都是2^(i-1)
6.完全二叉树的叶子结点值可能在层次最大的两层上出现
7.完全二叉树的右分支最大层层次为l则左分支下最大层必定为l或者l + 1
8.具有n个结点的完全二叉树的深度为 k = log2(n)下取值+ 1
9. 结点为n的完全二叉树编号，任意结点i有i = 0 无双亲，i > 1 双亲是结点 i/2的向下取值, 2i>n i无左孩子， 2i+1 > n则结点无右孩子，否则右孩子为2i+1
10.完全二叉树结点 = 分支结点 + 叶子结点  分支结点 =  结点数/2的向下取值

选择题

  1. 树转换成二叉树，因为转换的规则唯一转化为唯一二叉树
  2. 3个节点构造出5中不同的二叉树
  3. 完全二叉树的的结点数为1001求 叶子结点（设叶子结点数为n0,度为1的结点数为n1，度为2的结点个数为n2，可得 n0 + n1 + n2 = 10001 由二叉树性质得n0 = n2 + 1,推出 2n2+ 1 +n1 = 10001，对于完全二叉树来说度为1的点只可能为0或者1，n1 = 1 || 0 -> n2 = 500）
  4. 求二叉树的高度，每一层只有一个结点高度最高为n， 最低为完全二叉树log2(n)+1
  5. 深度为k的满m二叉树共有m^k-1个结点，第h层有m^(h-1);
  6. 树的二叉链表存储结构就是树的孩子兄弟表示法，链表中结点的左指针指向该结点的第一个孩子节点，右指针指向下一个兄弟结点，树的根节点没有兄弟节点，跟的右指针为空
  7. 树的存储结构有3种，双亲表示法，孩子表示法，孩子兄弟表示法，孩子兄弟表示法最常用，任何一颗树都能通过孩子兄弟表示法转化为二叉树进行存储
  8. 哈弗曼树中没有度为1的结点，只有度为0和度为2的结点
  9. 线索二叉树利用二叉链表的空链域来存放结点新的前驱和后继信息
  10. 二叉链表作为存储结构时，只能找到结点的左孩子和右孩子，不能直接得到结点在任一序列中的前驱和后继信息，这种信息只能在遍历中得到，引入线索二叉树来保存这种动态过程中得到的有关前驱和后继的信息
  11. 森林和二叉树的转换 ： 若F中有t个终端节点,n为终端结点，则结点总数为N = t + n 左右指针域的个数为 N左 = N右 = t + n，其中非空域 = 分支个数 = N分 = N - 1 = t + n - 1
  12. 森林转化成二叉树的规则为左孩子右兄弟，因为有n个非终端结点，有且仅有n个节点有孩子，转化为二叉树后则有n个非空的左指针域 N左非 = n
  13. 非空的有指针域的个数 N右非空 = N非空 - N左非空 = t + n - 1 - n = t - 1
  14. 空的右指针域的个数为N右空 = N右 - N右非空 = t + n -（t-1） = n + 1
  15. 哈夫曼树在构造是，重复在森林f中选择两个根结点权值最小的树作为左右子树，然后在继续构造一个新的二叉树，新的二叉树的根节点的权值为左子树和右子树权值之和

二叉树算法设计(二叉链表做诶二叉树的存储结构)

`1.统计二叉树的叶结点个数

int LeafNode(BiTree T){
    if (T == NULL)   return 0;
    else if (T->lchild == NULL && T->rchild == NULL)  return 1;
    else    return LeafNode(T->lchild) + LeafNode(T->rchild);
}
