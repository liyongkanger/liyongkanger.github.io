Ajax和json

同步和异步的概念和区别：
1.同步：客户端必须等待服务器的响应，在等待客户端不能做别的操作
2.异步：客户端在等待服务器端的响应过程中，可以做其他操作，ajax不更新整个页面的前提下完成与后台数据的获取

1.原生Ajax方式了解即可用的不多
2.JQeuryf实现Ajaxy异步实现：

    3.逻辑值
