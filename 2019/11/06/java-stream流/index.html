java 流

1.java.io包中定义了多种流型类，输入流输出流，字节流，字符流，节点流和处理流
例如： 输入流字节流InputStream 输入流字符流Reader 输出流字节流 OutputStream 输出流字符流Writer
2.输入流和输出流都是在程序的角度上，处理单位的不同字节流010101，字符流一个字符字符的读2个字节
3.节点流直接接在数据源上，在外面套在节点流一样是处理流
