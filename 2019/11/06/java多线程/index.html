java多线程

1.线程是一个程序内部的不同的执行路径
2.进程是一个静态概念，同一时间cpu只执行一个线程

2种方法创建新的线程
第一种： 定义线程类实现Runnable接口
Thread myThread = new Thread(target)//targ为Runable接口类型
Runnable接口一直一个方法：
public void run()：永凯定义线程的运行
使用Runnable接口为多个线程共享数据
在实现Runnable接口的run方法定义中可以使用Thread的静态方法
public staticThread CurrentThread()获取当前线程的引用
第二种：
可以顶一个一Thread线程的子类重写run方法如：
class MyThread extends Thread{
public void run(){
…
}
然后生成该类对象 MyThread myThread = new Thread();
}

线程转换

  1. start()去排队，不能一开始就直接调用，cup调入进去才开始执行
  2. 控制线程的基本方法：
    isAlive() 判断线程是否还活着，线程是否终止
    getPriority()获得线程点的优先级数值
    setPriority()设置线程的优先级数值
    Thread.sleep()指定当前显示睡多久
    join()调用某线程的该方法，将当前线程与该线程合并，等待该线程结束，在恢复到当前线程的运行
    yleld()让出cpu当前线程进入就绪队列等待调度
    wait()当前线程进入对象的Wait pool
    notify()/notifyAll()唤醒某个waite pool中的一个/所有等待线程
  3. sleep方法，睡眠指定的毫秒数，可能会抛出异常，可能睡着被打断就会抛出异常
